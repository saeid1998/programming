Write a program that reads the age of the candidates of the Islamic Council from the input and prints the age of the oldest in the output. Your program will continue to read from the input until a negative number of one is entered. As soon as this number appears in the input, the program should stop reading the input and print the largest age. The age of people ranges from 10 to 90 years.

Sample input:

17
15
39
51
14
32
28
-1

Sample output:

51
